# mmo-rpg-phaser

## Description 

Web development class project that consists in a simple MMO RPG game in JavaScript using Phaser. This project were made based in a tutorial avaliable at (https://phasertutorials.com/how-to-create-a-phaser-3-mmorpg-part-1/). We used this tutorial to build up the server and the basics structures for the game, but many new features were added by us, so as new gameplay mechanics, graphics, animations and a new map.
